## Run Locally

Clone the project

```bash
  git clone https://github.com/RudraG4/digital-business-card
```

Go to the project directory

```bash
  cd digital-business-card
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
